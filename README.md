# Gridlock — Round 2 (Idea / Proposal)

This repository contains an idea and proposal for Gridlock — Round 2, a challenge organized by Flipkart on the HackerEarth platform. It is intended as a conceptual and documentation repository and is not a runnable or deployable project.

## Overview
Gridlock is a competitive challenge that emphasizes algorithmic problem solving and optimization. This repository captures concept notes, design sketches, algorithmic approaches, and planning artifacts developed for the second round of the competition.

## Status
- Type: Idea / Proposal repository  
- Intended use: Documentation, brainstorming, and design reference  

## What’s in this repository
- docs/       — Design notes and problem analysis  
- proposals/  — Idea drafts and approach sketches  
- references/ — Research materials and links  
- examples/   — Small illustrative artifacts (conceptual)
